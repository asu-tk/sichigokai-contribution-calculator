# 七護會 任意献金目安額 計算ツール

消防団の既存Excel出欠表、または共有済みGoogleスプレッドシートURLをブラウザ内で読み込み、点検・訓練・火災出動の回数から、団員ごとの活動貢献率と「七護會 任意献金目安額」を算出する静的Webアプリです。

公開URL:

https://asu-tk.github.io/sichigokai-contribution-calculator/

アップロードされたExcelや計算結果はサーバーへ送信・保存しません。GoogleスプレッドシートURLから読み込む場合も、ブラウザ内でExcel形式として取得して計算します。DB、Supabase、ログイン機能は使いません。

## セットアップ

```bash
corepack enable
pnpm install
pnpm run dev
```

ブラウザで表示されたローカルURLを開きます。

## ビルド

```bash
pnpm run build
```

ビルド結果は `dist/` に出力されます。

## 公開方法

### GitHub Pages

`main` ブランチにpushすると、`.github/workflows/deploy-pages.yml` によりGitHub Actionsでビルドし、GitHub Pagesへ公開します。

このアプリは `vite.config.ts` で `base: './'` にしているため、GitHub Pagesのサブディレクトリ公開に対応しています。

初回だけGitHubのリポジトリ設定でPagesの公開元を有効化してください。

- 推奨: `Settings` > `Pages` > `Source` を `GitHub Actions` にする
- 代替: `Settings` > `Pages` > `Source` を `Deploy from a branch` にして、Branchを `main`、Folderを `/docs` にする

`docs/` にはビルド済みの静的ファイルを配置しています。

### Vercel

1. Vercelでこのフォルダをプロジェクトルートとして指定します。
2. Framework Preset は `Vite` を選びます。
3. Build Command は `pnpm run build`、Output Directory は `dist` を指定します。

## 対応Excel形式

- 月別シート構成に対応します。
- シート名は `4月`、`５月`、`11月`、`１１月` のような表記を想定しています。
- 全角数字は半角数字に変換して月判定します。
- 前期は4月〜9月、後期は10月〜3月、年間合計は両方を集計します。
- Excelファイルの直接アップロード、またはGoogleスプレッドシートURLからの取り込みに対応します。
- GoogleスプレッドシートURLは、共有設定を「リンクを知っている全員が閲覧可」にしたURL、または「ウェブに公開」したURLが対象です。
- 非公開のGoogleスプレッドシートは、ログイン機能のない静的Webアプリからは読み取れません。Excel形式でダウンロードしてアップロードしてください。

各月シートの想定構造:

| 行 | 内容 |
| --- | --- |
| 1行目 | タイトル |
| 2行目 | 活動名 |
| 3行目 | 種別。「点検」「訓練」「火災」の列のみ対象 |
| 4行目 | 月日 |
| 5行目 | 人員合計 |
| 6行目以降 | 団員別出欠 |

- A列はNO、B列は団員名、C列以降が活動ごとの出欠欄です。
- 「計」列、空欄列、3行目に対象種別がない列は無視します。
- B列が空欄の行は無視します。
- 出欠欄は数値 `1` または文字列 `"1"` のときだけ出席・出動扱いにします。

## 計算ルール

初期設定:

- 点検: 1回1ポイント
- 訓練: 1回2ポイント
- 火災出動: 1回3ポイント
- 火災ポイント上限: 半期ごとに10ポイント
- 国標準年額報酬: 36,500円（参考表示）
- 調布市団員年額報酬: 106,800円
- 七護會年会費: 15,000円
- 献金算定基準額: 90,000円
- 半期算定基準額: 45,000円
- 端数処理: 5,000円単位で四捨五入

計算式:

- 半期基本活動ポイント = 対象期間内の点検回数 × 点検P + 訓練回数 × 訓練P
- 個人獲得ポイント = 点検出席回数 × 点検P + 訓練出席回数 × 訓練P + 火災出動回数 × 火災P
- 火災ポイントは半期ごとに上限を適用します。
- 年間合計では前期・後期それぞれに火災上限を適用してから合算します。
- 活動貢献率 = 個人獲得ポイント ÷ 基本活動ポイント × 100
- 活動貢献率が100%を超える場合は100%表示にします。
- 基本活動ポイントが0の場合、活動貢献率は0%です。
- 差引後基準額 = 調布市団員年額報酬 - 七護會年会費
- 差引後基準額は、実際の報酬から七護會年会費を差し引いた参考金額です。
- 七護會 任意献金目安額の計算には、差引後基準額ではなく献金算定基準額を使用します。
- 半期算定基準額 = 献金算定基準額 ÷ 2
- 七護會 任意献金目安額 = 半期算定基準額 × 活動貢献率ごとの割合
- 金額は、設定した端数処理単位（1,000円または5,000円）で四捨五入します。

七護會 任意献金目安額:

| 活動貢献率 | 半期算定基準額に対する割合 |
| --- | ---: |
| 80%以上 | 0% |
| 60%以上80%未満 | 25% |
| 40%以上60%未満 | 50% |
| 20%以上40%未満 | 75% |
| 20%未満 | 100% |

例:

- 調布市団員年額報酬が106,800円、七護會年会費が15,000円の場合、差引後基準額は91,800円です。
- 運用上分かりやすくするため、献金算定基準額は90,000円や80,000円に変更できます。
- 献金算定基準額90,000円の場合、半期算定基準額は45,000円です。

## 出力

- CSVダウンロード
- Word（docx）ダウンロード
- 印刷用表示

Wordには、タイトル、作成日、対象期間、計算ルール、サマリー、団員別集計表、注意書きを出力します。

注意書き:

> 本資料に記載された金額は、七護會への任意献金を検討するための目安であり、確定事項ではありません。

## 注意事項

- このツールは任意献金を検討するための計算機です。
- 団員名簿や個人情報を保存・管理しません。
- Excelファイルはブラウザ内で読み取り、計算結果もブラウザ内だけで表示します。
- GoogleスプレッドシートURL取り込みは、共有/公開されたシートだけに対応します。非公開シートを扱う場合は、認証付きの別システムが必要です。
- 公開時は静的ファイルとして配信できます。
